Some further examples of Bootstrap in usage alongside some calculating logic done via React.

'Roll the Dice' allows the user to modify the number of dice to be rolled as well as the sides of the dice.

'Grades Calculator' allows the user to calulate the grades of various courses: if the courses have the same name, they are bundled as one; if they have a different name, they are
calculated separately.

All React script work is left inside the .html files for simplicity's sake.
